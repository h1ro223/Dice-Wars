# 第6作 サイコロ・ウォーズ v1.4

某ソシャゲのイベント内ミニゲームが面白かったので再現しました
サイコロの出目で相手にダメージを与えたり、シールドを張ったりして戦うゲームです。
先にHPを0にした方が勝ち！

急に話変わるんだけど最近　崩壊：スターレイル　ってゲーム始めたんだけど面白いね

開発はGoogle Antigravityにて Gemini・Claude(Opus 4.6 Sonnet 4.6 Thinking)・GPT-OSS 120B (Medium) の3モデルを使用しています。
またv1.4から実装したオンラインプレイは Renderというクラウドサービスを用いてサーバー運用しております。

Render
https://render.com/ 

※サーバーに接続時、1分近くかかる場合がありますが、現在はRenderを無料プランで運用しており
その仕様で15分アクセスがないとサーバーがストップし、再起動まで1分かかるためです。1度起動すれば
ストップするまでは接続は一瞬で出来ます。

無料プランのため、リソース制限もあるため突然オンラインプレイが出来なくなる場合がございます。
ご了承下さい。

# 不具合のお知らせ

オンラインプレイをv1.4にて実装しましたが、ゲームプレイ時高確率で
ホストとゲストそれぞれのゲーム情報が同期されず意図しないバグが発生することを確認しています。

例
1.先行と後攻を決める画面で、通常だと 1Pが先攻(攻撃側)・2Pが後攻(防御側) またはその逆が正しい動作だが
割と高確率で 1Pと2Pが同時に先攻(攻撃側)になってしまう

2.お互いの画面で1Pと2Pのキャラカードの情報が逆に表示されてしまっている。
ホスト(1P)なら下が1P:ガルム、上が2P:ホースラと表示されるが
ゲスト(2P)だと下が1P:ホースラ、2P:ガルムと表示されてしまい、体力やダメージ、ATK・DEF情報に
ズレが出てしまう。

などです。
