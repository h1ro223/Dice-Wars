# Antigravity インストール済み拡張機能一覧

別のPCで環境を再現する際は、各拡張機能の「ID」を拡張機能の検索欄に入力してインストールしてください。

## 🌐 Web開発関連 (HTML/CSS/JS)
* **Live Server** (ID: `ritwickdey.liveserver`)
  * ローカル開発サーバーを起動し、ファイルの変更をリアルタイムでブラウザに反映（ライブリロード）します。
* **Live Preview** (ID: `ms-vscode.live-server`)
  * エディタ内で直接ブラウザのプレビューを確認できるMicrosoft公式の拡張機能です。

## 🐍 Python環境
* **Python** (ID: `ms-python.python`)
  * Python開発の基本サポート機能。
* **Python Debugger** (ID: `ms-python.debugpy`)
  * Pythonのデバッグ機能。
* **Python Environment Manager** (ID: `ms-python.vscode-python-envs`)
  * 仮想環境などの管理。
* **Pyrefly** (ID: `meta.pyrefly`)
  * コード解析・リファクタリングツール。

## ☕ Java環境
* **Extension Pack for Java** (ID: `vscjava.vscode-java-pack`)
  * 以下の機能が一括でインストールされます：
    * Language Support for Java(TM) by Red Hat (`redhat.java`)
    * Debugger for Java (`vscjava.vscode-java-debug`)
    * Test Runner for Java (`vscjava.vscode-java-test`)
    * Maven for Java (`vscjava.vscode-maven`)
    * Project Manager for Java (`vscjava.vscode-java-dependency`)
* **Gradle for Java** (ID: `vscjava.vscode-gradle`)

## 🐘 PHP環境 (DEVSENSE)
* **PHP Tools for VS Code** (ID: `devsense.phptools-vscode`)
* **IntelliPHP** (ID: `devsense.intelli-php-vscode`)
* **PHP Profiler** (ID: `devsense.profiler-php-vscode`)
* **Composer** (ID: `devsense.composer-php-vscode`)

## 🔧 その他のプログラミング言語
* **Go** (ID: `golang.go`)
* **clangd (C/C++)** (ID: `llvm-vs-code-extensions.vscode-clangd`)
* **Ruby LSP** (ID: `shopify.ruby-lsp`)

## 🛠️ 共通・ユーティリティ関連
* **Japanese Language Pack for Visual Studio Code** (ID: `ms-ceintl.vscode-language-pack-ja`)
  * エディタの日本語化パック。
* **Discord Rich Presence** (ID: `icrawl.discord-vscode`)
  * 状況をDiscordのステータスに表示する機能。
* **AG Quota** (ID: `henrikdev.ag-quota`)
  * Antigravity用の専用ユーティリティ。
